# Week 5 – Advanced Data Manipulation with Pandas
## Customer Sales Analysis

## Project Overview
This project is part of Week 5 of the Developers Arena Data Science Internship.
The goal is to perform advanced data manipulation using Pandas to analyze
customer purchasing patterns and sales performance.

The analysis focuses on revenue trends, regional performance, customer churn,
and identification of top customers.

---

## Setup Instructions
1. Install Python (3.8 or above)
2. Install required libraries:
   pip install -r requirements.txt
3. Open and run:
   customer_analysis.ipynb

---

## Project Structure
customer_analysis.ipynb  → Main analysis notebook  
sales_data.csv           → Sales dataset  
customer_churn.csv       → Customer churn dataset  
analysis_report.pdf      → Final analysis report  
requirements.txt         → Dependencies  

---

## Analysis Performed
- Data cleaning and preparation
- Grouping and aggregation
- Date-based analysis
- Dataset merging
- Pivot table creation
- Visualization dashboard

## Visualizations Created
