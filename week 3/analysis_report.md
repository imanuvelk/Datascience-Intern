# Sales Data Analysis Report

## Overview
This report analyzes a simple sales dataset using Python and pandas.

## Key Metrics
- Total Sales calculated using sum
- Average Sales calculated using mean
- Highest Sale identified using max
- Best Selling Product identified using groupby

## Data Cleaning
Missing values in Quantity and Total_Sales columns were handled using fillna.

## Insights
- Notebook generated the highest total sales
- Sales data is now clean and ready for further analysis
