# Week 4 – Data Visualization & Complete Data Analysis Project

## Project Overview
This project performs a complete data analysis workflow on e-commerce sales data,
including data cleaning, analysis, and visualization using Python.

## Setup Instructions
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py

## Folder Structure
data/ → Dataset  
visualizations/ → Generated charts  
report/ → Analysis report  
main.py → Main analysis script  

## Charts Created
- Bar chart: Sales by Product
- Line chart: Monthly Sales Trend

## Insights
- Identified best-selling product
- Observed monthly sales patterns
